    Why#

    I started this blog to practice my writing skills and to document my personal projects. I plan to continue to update this blog with lessons learned from the projects I work on, along with documentation so others can learn from knowledge I have compiled. I hope that, by writing down and double checking my knowledge, I will gain a greater understanding of the topics I write about.

    How#

    I set up this blog using Hugo and used GitHub Pages for hosting. I also using multiple git repos to make the project management easier. The following will be a step-by-step guide on how to set up a blog like this one.

    Tools#

    Hugo is a website building tool that takes markdown files and spits out static HTML and CSS. It is written in go and has a community which makes themes that you can download. Git is a piece of version control software not to be confused with GitHub, which is a website where you can store project managed by the git. I would suggest reading through this guide if you don’t have a working knowledge of git. Visual Studio Code is a free Text-Editor/Lite-IDE that we will use to edit the markdown and other files that will build this blog.

    Windows Tool Setup#

    I would suggest using Chocolatey on windows as it makes setting up and managing dev tools a bit easier on Windows. If you do not have Chocolatey already, install go here and follow the deration to get it installed on your computer. With Chocolatey installed, run the following commands, one after another, in an administrator PowerShell/CMD window. This will install all the needed tools.

    choco install git
choco install vscode
choco install hugo

    Mac Tool Setup#

    For Mac, I suggest using Homebrew for the same reasons I suggest using Chocolatey on Windows. To install homebrew on your mac (m1 macs too), run the following command:

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    With Homebrew installed, you can run the following commands:

    brew install hugo
brew install git

    To install VS Code, go here to download the newest version, then move the .app package into your application folder.

    Git Setup#

    Before setting up the project with Hugo, a few things need to be setup. If you do not already have one, you should make a GitHub account here. To configure git to work with your GitHub account, we first need to set the username and email to match the ones of your GitHub account. To set the username and email of git, run the following commands:

    git config --global user.name "Your username"
git config --global user.email "Your email"

    Git can connect to GitHub by HTTPS or SSH but will connect through HTTPS by default. All you need to do is put in your GitHub password when pushing. However, GitHub plans to remove support for HTTPS at some point, as it is less secure, so follow this guide to setup git to connect to GitHub by SSH.
